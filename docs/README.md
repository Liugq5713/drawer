### prop

| 参数          | 说明         | 类型             | 可选值 | 默认值 |
| ------------- | ------------ | ---------------- | ------ | ------ |
| controls      |              | String/Array     |        |        |
| controlConfig | 控件的位置   |                  |        |        |
| controlOffset | 控件的偏移   | String/Number    |        |        |
| show          | 是否展示抽屉 | Boolean/Function |        |        |
